move_random.is_front_hazard_active ( hazards)

Checks if a hazard is detected in front. 

   This function iterates through the detections in the given hazards object and checks
   if any hazard is detected in the front. It returns True if a hazard is detected,
   otherwise returns False.
Parameters
hazardsA hazards object containing information about detected hazards.
Returns
bool True if a hazard is detected in front, False otherwise.
Note
The function assumes that the hazards object has a 'detections' attribute, which is expected to be an iterable containing information about individual detections.

The function uses the value of 'detection.type' to determine if a hazard is detected. A value of 0 represents no hazard, and any other value represents a hazard.

Examples

>>> hazards = get_front_hazards()
>>> is_front_hazard_active(hazards)
True
